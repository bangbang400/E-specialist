# E-specialist
<h4>英単語学習IOSアプリ　開発中</h4>

開発中

<h2>実施したこと</h2>

・FirebaseDBよりデータ取得し、リストへ英単語の情報を表示させる

・カスタムセルを使用してボタンを単語を一覧表示した際に「わかる」・「わからない」の判定を行うボタンを配置
